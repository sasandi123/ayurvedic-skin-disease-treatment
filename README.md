## Project Description

An AI-powered system that provides Ayurvedic treatment recommendations for common skin conditions using machine learning and image analysis, specifically tailored for Sri Lankan "Hela Wedakama" practices.

---

## Key Features

- **Image-based skin disease detection** - Upload skin condition images for analysis
- **Symptom preprocessing with NLP** - Natural language understanding of symptoms
- **Safety triage system** - Emergency detection and risk assessment
- **Disease classification** - Ensemble ML + CNN models for accurate diagnosis
- **Personalized herb recommendations** - Sri Lankan Ayurvedic herbs and formulations
- **Hela Wedakama integration** - Traditional Sri Lankan Ayurvedic practices
